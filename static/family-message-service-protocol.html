1.1 亲情短信服务：指国科服务依法为监狱服刑人员亲属提供的文字短信服务，监狱服刑人员亲属在遵守本协议及国科服务相关协议、政策及规则的前提下，可以通过该服务与服刑人员实现文字沟通交流。该服务为有偿服务，需用户额外付费购买。

1.2 用户：指已成功开通并购买本服务，在服务有效期内享受本服务的监狱服刑人员亲属。前述“亲属”的具体范围以服刑人员所在监狱规定为准。

1.3 亲情短信服务协议：指本协议。本协议的签署及履行，须通过您的国科服务账号进行，因此，《国科服务用户个人信息保护政策》《国科服务用户注册协议》及相关附属协议、政策及规则对您具有约束力。本协议未约定部分，按照前述文件执行；本协议与前述文件冲突的，以本协议为准。

1.4 亲情短信服务规则：指国科服务已经或未来不时发布的与本服务相关的协议、公告、页面说明、通知、FAQ、优惠活动等内容。前述内容作为本协议的组成部分，与本协议冲突的，以发布时间在后者为准。

二、 服务内容

2.1 您同意开通亲情短信服务。我们提请您注意，亲情短信服务的用户资格及审核规程等环节均由监狱管理部门规范，国科服务将按照法律法规规定及监狱管理部门的要求向您提供本服务。

2.3 国科服务将为监狱管理部门开通后台账号，以便监狱管理部门能够依法审查您与监狱服刑人员之间的亲属关系，并最终决定您是否具备购买本服务的资格。

2.4 您在使用本服务期间，监狱管理部门可通过后台账号依法审查您向监狱服刑人员发送的短信内容，短信内容将在审核通过后发送给监狱服刑人员。同样地，监狱服刑人员向您发送的短信内容亦将在审核通过后发送给您。

2.5 为了进一步改善用户体验，国科服务将会持续配合监狱管理部门改进本服务，为用户提供版本升级、功能升级等服务，适时对本服务的功能进行增加、删减、更新、调整，或取消本服务类型，具体以您使用服务当时国科服务相关页面的展示为准。

3.1 收费方式：本服务采用先收费后服务的方式，用户可通过国科服务账户余额完成亲情短信服务费用的支付。（注：还有别的支付方式吗？比如直接通过支付宝/微信/银行卡支付？请勿漏项）

3.4 费用退还：

（1） 按短信条数付费形式中，发送短信及接收短信均单独计费。发送短信时，费用支付成功并不代表短信已成功发送至监狱服刑人员，如短信审核未通过，该条短信费用将退回至您国科服务的账户中。接收短信时，该条短信费用将直接从您国科服务账户余额内扣除。

（2） 包月付费形式中，当月收发短信的费用均包含在包月服务费中，收发条数不受限制。您理解并同意，包月服务费并非预付款、存款、定金或储蓄卡等性质，包月服务费一旦支付成功，即视为您在包月服务期内持续使用本服务，无特殊情况，国科服务将不接受任何形式的退费要求。
