一、注销账户的条件

如果您仍执意注销国科服务账户，请您确保账户同时满足以下条件：

1. 您的账户处于正常、有效使用状态；

2. 您自愿放弃账户在国科服务系统中的资产和虚拟权益（包括但不限于账户余额、账户权益等），并确保账户无欠款；

3. 账户内无未完成的订单和服务；

4. 账户无任何纠纷，包括与第三方的争议或纠纷、投诉举报或被投诉举报；

5. 账户已经解除与其他网站、其他APP的授权登录或绑定关系。

如您申请注销账号时有如上情形，可能无法自助完成注销流程，您可以通过拨打热线电话的方式联系我们由人工客服予以处理。

二、注销账户的后果

1. 账号注销具有不可逆转和不可撤销性，国科服务账户一旦被注销将不可恢复，请您在操作之前自行备份国科服务账户相关的所有信息和数据。如果您在注销后以相同手机号码或相同电子邮箱再次向我们申请账号，此时该账号将默认为新的用户账号。

2. 在国科服务账户注销期间，如果您的国科服务账户涉及争议纠纷，包括但不限于投诉、举报、诉讼、仲裁、国家有权机关调查等，国科有权自行终止本国科服务账户的注销而无需另行得到您的同意。

3. 注销国科服务账户，您将无法再登录或使用本国科服务账户，也将无法再检索、访问、获取、继续使用和找回您国科服务账户中及与账户相关的任何内容或信息，包括但不限于：

（1）该账号下的您自主设置的个人资料（如：头像、昵称等）；

（2）该账号下您主动提供或在使用过程中授权我们收集的个人信息（如：绑定的手机号、邮箱等）；

（3）其他与该账号相关的内容、信息、数据、历史记录或服务记录等。

4. 注销您的国科服务账户并不代表此账户注销前的账户行为和相关责任得到豁免或减轻。

5. 账号一旦注销，您将不再享有您与我们签署的相关用户协议、其他权利义务性文件中约定的适用于注册用户的权利，但法律法规、规章、规范性文件，政府的政策、命令等另有要求或用户服务协议中另有规定的除外。

三、其他须知

1. 您在点击本协议下方的“确认”按钮即视为您完全接受本《注销须知》并进行注销流程，在点击之前，请您再次确认已知悉并完全理解本注销的全部内容。

2. 注销申请提交后，您将不能再通过该账号使用国科服务产品及服务。我们处理完成您的注销申请后，我们的客服团队将有可能通过您留下的联系方式就注销申请的结果与您联络。
